Uma dica muito importante é criar primeiramente o html da página, e depois inserir os estilos css.
Por exemplo:
    1. Criar html do menu superior.
    2. Criar css do menu superior.
    3. Criar html do corpo.
    4. Criar css do corpo.
    5. Criar html do footer.
    6. Criar css do footer.
